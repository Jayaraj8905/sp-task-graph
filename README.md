 
# How to Start

$ Install the latest node

$ git clone https://github.com/Jayaraj8905/sp-task-graph.git

$ cd sp-task-graph

# Install the project's dependencies
$ npm install

# Start the Server
$ npm start

Application will open in the below url
http://localhost:3000/
